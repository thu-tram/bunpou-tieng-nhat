# 〜のに、〜けど、〜は〜が、〜は（対比）


## 〜のに
〜けれど   「当然〜だろう」と予想される事柄とは違った結果になったことを表す。
話の手の驚きや不満などの気持ちが含まれる。

V（普通形） + のに イAい + のに ナA + のに N +　なのに

薬を飲んだのに、全然良くなりません。

昨日、早くたくさん寝たのに、まだ眠いです。

トムさんは２年日本へ留学したのに、全然日本語が話せません。

## 〜のに
物の用途を述べるときに使う表現。動詞は「使います」を使うことが多い。

V（辞書形） + のに N + に

風呂敷は物を包むのに使います。

この箸はパスタを食べるのに使います。

これはおにぎりを作るのに使います。

## 〜のに
物や場所などを評価するときに使う表現
V（辞書形） + のに N + に

田舎は買い物するのに不便です。

東京は買い物するのに便利です。

東京は電車が多いので、会社に通うのに便利です。
